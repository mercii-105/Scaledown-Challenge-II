About EcoFAQ

EcoFAQ is a lightweight environmental information assistant designed to deliver compressed sustainability knowledge quickly. It uses structured FAQs with keyword matching, category filtering, and quick suggestions to simulate a helpful chatbot experience.

Core Features

  • FAQ knowledge base with categories + tags
  • Quick & Detailed answer modes (compressed responses)
  • Search + suggestions
  • Related question recommendations
  • Clean modern UI with light theme
